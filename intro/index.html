hexo+github

发表于 2022-01-13 更新于 2022-02-24 分类于 website

利用hexo+github制作个人博客

新建仓库

新建一个仓库，名称为username.github.io，此处的username要与前面的用户名保持一致，并且必须配置SSH链接，要不然hexo d会失败。

安装环境

安装Node.js

在官网下载Node.js,配置好环境变量，能使用npm命令

安装Hexo

安装命令：

npm install -g hexo-cli
安装完成之后配置好环境变量，能使用hexo命令

初始化、生成并部署项目

1. hexo init {name}
2. 进入name文件夹，输入命令如下
hexo generate
生成了静态网页
3. 本地运行，命令如下：
hexo s
可以在本地查看博客站点了
4. 部署项目，命令如下：
hexo deploy
项目就可以通过自己的username.github.io访问

增加项目

新建文章

hexo new helloworld

标签页

hexo new page tags

分类页

hexo new page categories

修改主题的_config.yml


去掉tags和categories前面的“#”即可添加标签页和分类页
